View on GitHub Quick Start

🛠 Three tools, one mission

🎵

data_to_melody.py

Converts daily power and water savings into a MIDI melody. Supports CSV/JSON input, batch processing, Tibber and Home Assistant formats, piano-roll preview, and FluidSynth playback.

🔥

genre_fusion.py

Algorithmically blends classical, jazz, rock, electronic, blues, and ambient styles into a two-track MIDI composition with per-genre rhythm patterns and chord voicings.

🎼

song_builder.py

Combines a metrics-driven melody, bass line, and genre-fusion harmony layer into one complete multi-track MIDI file — your daily Symphony of Order.

🚀 Installation & Quick Start

# Clone and install
git clone https://github.com/GizzZmo/JonArveSymphonyOfOrder.git
cd JonArveSymphonyOfOrder
pip install -r requirements.txt
# Generate a melody from today's savings
python data_to_melody.py --power 7.8 --water 1850 --scale pentatonic

# Blend genres into a fusion track
python genre_fusion.py --genres jazz blues ambient --bars 32

# Build a full 3-track song
python song_builder.py --power 9.1 --water 2100 --genre jazz --scale dorian
# Load data from a CSV and preview without writing a file
python data_to_melody.py --csv data.csv --dry-run

# Batch-generate one MIDI per day in a date range
python data_to_melody.py --csv data.csv --batch --start 2024-01-01 --end 2024-01-31

# Run the web form locally
python app.py
