# Examples

This directory is the home for example MIDI output files generated by the
JonArve Symphony of Order tools. The files are not committed to the repository
(they are binary MIDI data), but you can generate them yourself with the
commands below.

---

## How to generate examples

Make sure you have installed the dependencies first:

```bash
pip install -r requirements.txt
```

### Example 1 — Pentatonic melody from moderate savings

```bash
python data_to_melody.py \
    --power 7.8 --water 1850 \
    --scale pentatonic --tempo 92 --bars 16 \
    --output examples/melody_pentatonic.mid
```

### Example 2 — Jazz-blues fusion, 32 bars

```bash
python genre_fusion.py \
    --genres jazz blues \
    --bars 32 --tempo 108 \
    --output examples/fusion_jazz_blues.mid
```

### Example 3 — Full 3-track song with dorian melody

```bash
python song_builder.py \
    --power 12.5 --water 2400 \
    --genre electronic --scale dorian \
    --bars 24 --tempo 120 \
    --output examples/song_electronic_dorian.mid
```

---

## What the files sound like

| File | Character |
|------|-----------|
| `melody_pentatonic.mid` | Bright, hopeful lead line — no dissonant intervals |
| `fusion_jazz_blues.mid` | Swinging, soulful two-track harmony with ghost notes |
| `song_electronic_dorian.mid` | Driving 3-track arrangement — bass + melody + arpeggios |

Open any `.mid` file in a DAW (Reaper, GarageBand, LMMS) or a MIDI player to
hear your planetary savings turned into music.
