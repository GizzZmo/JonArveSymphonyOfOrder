# JonArveSymphonyOfOrder

**JonArveSymphonyOfOrder** transforms raw data of power consumption and water-saving metrics into generative melodies — sonic alchemy that turns kilowatts saved and liters of clean water protected into audible beauty.

## Files

| File | Description |
|------|-------------|
| `data_to_melody.py` | Converts daily power/water savings into a MIDI melody |
| `genre_fusion.py` | Algorithmically blends classical, jazz, rock, and electronic styles |
| `block_breaker_journal.md` | Daily journal template for overcoming composer's block |

## Requirements

```
pip install midiutil
```

## Usage

### Generate a melody from your daily metrics

```bash
python data_to_melody.py
```

Edit the `power_saved_kwh` and `water_saved_liters` values at the bottom of `data_to_melody.py` to match your real metrics, then run the script each evening. The output MIDI file is your personal anthem of impact.

### Generate a genre-fusion composition

```bash
python genre_fusion.py
```

Blends classical, jazz, rock, and electronic styles into a single MIDI file (`fusion_order.mid` by default).

### Composer's block journal

Open `block_breaker_journal.md`, copy it for today's date, and fill it in while the generated MIDI plays.
